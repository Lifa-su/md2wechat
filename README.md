# MD2WeChat — Markdown 转公众号排版工具

一款纯前端的 Markdown 转微信公众号排版工具，左右分栏实时预览，一键复制富文本直接粘贴到公众号编辑器。

## 功能

- **实时预览** — 左侧写 Markdown，右侧即时渲染公众号排版效果
- **4 套主题** — 简约 / 科技 / 文艺 / 商务，一键切换
- **代码高亮** — 基于 highlight.js，支持主流编程语言
- **富文本复制** — 自动内联样式，粘贴到公众号编辑器不丢格式
- **完整语法** — 支持标题、列表、表格、引用、图片、链接、删除线等
- **可拖拽分栏** — 拖动中间分隔条调整编辑区和预览区比例
- **数据安全** — 纯前端运行，内容不上传任何服务器

## 使用方法

### 方式一：直接打开

双击 `index.html` 用浏览器打开即可使用（推荐 Chrome）。

### 方式二：本地服务器

```bash
cd md2wechat
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

### 使用流程

1. 在左侧编辑区输入或粘贴 Markdown 内容
2. 右侧自动渲染公众号排版效果
3. 从顶部下拉框选择主题风格
4. 点击右上角「📋 复制富文本」按钮
5. 打开公众号后台编辑器，`Ctrl+V` / `Cmd+V` 粘贴

## 技术栈

- [marked.js](https://github.com/markedjs/marked) — Markdown 解析
- [highlight.js](https://github.com/highlightjs/highlight.js) — 代码语法高亮
- 依赖通过 CDN 引入，无需安装或构建

## 主题预览

| 主题 | 风格 | 适用场景 |
|------|------|----------|
| 简约 | 黑白为主，干净清爽 | 日常文章、通知公告 |
| 科技 | 紫色渐变，现代感 | 技术博客、产品介绍 |
| 文艺 | 衬线字体，古典韵味 | 散文随笔、读书笔记 |
| 商务 | 蓝色系，专业稳重 | 行业分析、商业报告 |

## 注意事项

- 推荐使用 Chrome 浏览器，富文本复制兼容性最好
- 图片请使用外链 URL，公众号编辑器会自动上传到腾讯服务器
- 复制功能使用 Clipboard API，需要 HTTPS 或 localhost 环境（直接打开本地文件也可以）
